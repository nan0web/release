# Getting Started with @nan0web/release

## Вступ

`@nan0web/release` — це інструмент для управління життєвим циклом проекту через контракти (тести).

## Встановлення

Використовуйте pnpm для встановлення:

```bash
pnpm add @nan0web/release
```

## Перший крок: Ініціалізація

Щоб розпочати роботу з релізами, ініціалізуйте структуру:

```bash
release init v1.0.0
```

Це створить директорію `releases/1/0/v1.0.0/` з базовими контрактами.
