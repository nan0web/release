# Economy & Themes

## Universal Design Economy

Пакет `@nan0web/release` підтримує універсальну систему тем. Налаштування візуального вигляду Docs Site або CLI Sandbox можна експортувати та продавати у **Theme Store**.

### Модель підписки

Ви можете оформити підписку на преміум-теми, які гарантують:

- Підтримку нових компонентів релізу.
- Адаптацію під різні інтерфейси (Web/CLI).
- Режим **High Contrast** для максимальної доступності.

### Комісія

- **82%** отримує автор теми.
- **18%** утримує платформа NaN0Web.
